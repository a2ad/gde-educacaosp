Nesta seção você encontrará tudo relacionado à interface dos projetos.

A organizazação é feita por categorias e, cada uma possui módulos que são os principais componentes utilizados nos projetos. É possível visualizar cada módulo renderizado bem como seu código HTML.

Leia mais sobre como utilizar.

Suporte

Testado nos principais navegadores: Chrome, Firefoz, Opera (a partir da versão 15), Safari e Internet Explorer (a partir da versão 7).

Caso a caso

Vale lembrar que os módulos estão em constante atualização e que as tags utilizadas aqui são a cargo de exemplo. Pode ser que em seu projeto, para manter a ordem semântica, você tenha que usar por exemplo, um h3 ao invés de um h2. Cabe a você analisar caso a caso, e utilizar da melhor maneira possível.
