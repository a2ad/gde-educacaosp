Para poder utilizar os módulos de interface do Guia de Estilo, primeiro você precisa copiar o link da folha de estilos na sua página:

<link rel="stylesheet" href="http://www.educacao.sp.gov.br/gde/css/main.css">

Feito isso, escolha um módulo de sua preferência, copie o código HTML e cole na sua página.

Antes de usar

Dependências

Vale lembrar que a idéia inicial do projeto é fornecer um tema base cinza (link para download em breve) para utilização nos projetos, no entanto, caso você deseje apenas utilizar os módulos, talvez você precise fazer download de alguns itens, como descrito abaixo.

Em um mundo ideal não seria necessário nenhuma dependência, mas como o projeto utiliza uma fonte específica para a tipografia, além de utilizar o Font Awesome para iconografia, por questões de segurança, alguns navegadores bloqueiam a utilização da propriedade font-face quando a fonte está hospedade em outro servidor. Resumindo, ao linkarmos o CSS do Guia, a fonte Aller e os ícones que são utilizados nos módulos não funcionariam.

Uma solução seria editar o .htaccess do projeto e alterar algumas configurações do navegador, mas optamos por algo mais simples e, deixamos como download as dependências do projeto.

Passo a passo:

  1. Faça o download da pasta com as fontes e descompacte dentro da pasta de CSS do seu projeto, ou na sua pasta de preferência.
  2. Faça o dowload do arquivo CSS do Font Awesome. Caso seu projeto precise dar suporte para o IE7, você precisa fazer o download do arquivo específico para essa versão do navegador.
  3. No seu CSS insira a seguinte marcação: 
    @font-face {
   font-family: 'AllerBold';
   src: url('font/aller_bd-webfont.eot');
   src: url('font/aller_bd-webfont.eot?#iefix') format('eot'),
        url('font/aller_bd-webfont.woff') format('woff'),
        url('font/aller_bd-webfont.ttf') format('truetype'),
        url('font/aller_bd-webfont.svg#webfonti8hol64m') format('svg');
   font-weight: normal;
   font-style: normal;
}

@font-face {
   font-family: 'AllerRegular';
   src: url('font/aller_rg-webfont.eot');
   src: url('font/aller_rg-webfont.eot?#iefix') format('eot'),
        url('font/aller_rg-webfont.woff') format('woff'),
        url('font/aller_rg-webfont.ttf') format('truetype'),
        url('font/aller_rg-webfont.svg#webfontGBXJYuuM') format('svg');
   font-weight: normal;
   font-style: normal;
}

@font-face {
   font-family: 'FontAwesome';
   src: url('font/fontawesome-webfont.eot?v=3.2.1');
   src: url('font/fontawesome-webfont.eot?#iefix&v=3.2.1') format('embedded-opentype'),
   	  url('font/fontawesome-webfont.woff?v=3.2.1') format('woff'), 
   	  url('font/fontawesome-webfont.ttf?v=3.2.1') format('truetype'), 
   	  url('font/fontawesome-webfont.svg#fontawesomeregular?v=3.2.1') format('svg');
   font-weight: normal;
   font-style: normal;
}

    Vale lembrar que dependendo do local que você descompactou a pasta de fontes no seu projeto, talvez você precise alterar os caminhos das nas chamadas acima.

  4. Por fim, insira os links da folha de estilo referentes ao Font Awesome: 
    <link rel="stylesheet" href="css/font-awesome.css">
<!--[if IE 7]>
   <link rel="stylesheet" href="css/font-awesome-ie7.css">
<![endif]-->

Bugs?

Todos os módulos são testados diversas vezes, mas erros podem acontecer. Caso encontre algum, abra uma issue no GitHub do projeto, ou envie um e-mail para design@a2comunicacao.com.br com os detalhes do erro.
