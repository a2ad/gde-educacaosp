O que é?

Nova versão do guia de estilos dos projetos das coordenadorias e secretarias do portal da Educação de SP.

Como funciona?

São 3 categorias principais: Sobre, Conteúdo e Interface. Cada categoria contém assuntos específicos e, que podem ser acessados pelo menu lateral.

  • Sobre - Relacionada às informações e documentação do projeto.
  • Conteúdo - Diretrizes e guia para a publicação de conteúdo.
  • Interface - Relacionada à interface visual e principais componentes dos projetos. Cada componente é tratado como um módulo, onde é possível visualizá-lo além de consultar seu código HTML.

Por que?

A idéia é a padronização. Seja de conteúdo, elementos de interface ou código. Você pode ler um pouco mais sobre o assunto no prefácio.
